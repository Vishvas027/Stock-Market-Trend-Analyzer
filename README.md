# Stock Market Trend Analyzer 📈

A Python-based tool that uses **Moving Average Crossovers** to identify potential Buy and Sell signals in the stock market.

## 🚀 Features
- **Data Retrieval:** Uses `yfinance` to fetch real-time market data.
- **Trend Logic:** Implements 20-day (Fast) and 50-day (Slow) Simple Moving Averages (SMA).
- **Signal Detection:** Uses NumPy to detect "Golden Crosses" and "Death Crosses."
- **Visualisation:** Generates interactive charts using Matplotlib.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, YFinance
